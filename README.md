# PPD-GMFNet

## Description

This repository contains the source code for **PPDGMFNet**, a deep learning framework for postpartum depression (PPD) recognition in maternal subjects.

The model leverages data collected from **Huawei Band 8** wearable devices and integrates the **Edinburgh Postnatal Depression Scale (EPDS)** for enhanced PPD identification. The combination of physiological signals from wearable sensors and psychological scale features enables robust and interpretable screening of postpartum depression.

## Code Structure

The primary implementation is provided as **Jupyter Notebook files (`.ipynb`)**
